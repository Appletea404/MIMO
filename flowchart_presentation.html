⚗️ Expert Mode (기본)
전문가 제품 데이터 기반
1. 사용자 질문 → Okt 형태소 분석
2. TF-IDF 벡터화 → 코사인 유사도 계산
3. 피부타입·고민 매칭 시 +0.05 보너스
4. 점수 상위 1개 제품 추천
💬 Review Mode
올리브영 리뷰 데이터 기반
1. 리뷰 텍스트로 학습된 TF-IDF 모델 사용
2. 코사인 유사도 > 0 인 제품 필터링
3. 중복 제거 후 상위 3개 제품 반환
4. 실제 리뷰 기반 추천 (사용자 경험 반영)
