Cosine Similarity · 벡터 간 방향 유사도 · Expert Mode / Review Mode 추천 계산

코사인 유사도란?

두 벡터 사이의 각도(cosine)를 이용해 유사도를 측정하는 기법입니다. 벡터의 크기(단어 빈도의 절대값)가 아닌 방향(어떤 단어를 쓰는지의 패턴)을 비교하기 때문에, 문서 길이가 달라도 안정적으로 유사도를 계산할 수 있습니다.


본 시스템에서는 사용자 쿼리 벡터와 상담 질문 또는 리뷰 벡터 사이의 코사인 유사도를 계산합니다. Expert Mode는 가장 관련성 높은 상담 행 1개를 고르고, Review Mode는 관련 제품 리뷰를 최대 3개까지 순위화합니다.

핵심 역할: TF-IDF 벡터 → 코사인 유사도 점수 → Expert 1위 상담 선택 / Review 상위 3개 제품 추천

수식

cos(θ) = (A · B) / (‖A‖ × ‖B‖)
A · B = 두 벡터의 내적 · ‖A‖, ‖B‖ = 각 벡터의 크기(L2 norm)
유사도 범위: 0.0 (전혀 다름) ← → 1.0 (완전히 동일)
TF-IDF 벡터는 항상 양수 → 값 범위 [0, 1]

각도와 유사도

θ ≈ 0°
거의 동일한 방향
score ≈ 1.0
→ 매우 유사한 상담/리뷰
θ ≈ 45°
부분적으로 겹침
score ≈ 0.7
→ 관련 있는 상담/리뷰
θ = 90°
수직 (공통 단어 없음)
score = 0.0
→ 관련 없는 상담/리뷰

실제 구현 코드 (web_app.py)

from sklearn.metrics.pairwise import linear_kernel
import numpy as np

# ── Expert Mode ─────────────────────────────────────────────────
cleaned = preprocess_input(user_input)
similarities = linear_kernel(tfidf.transform([cleaned]), tfidf_matrix)[0]

# 프로필 보너스 (+0.05 × 일치 항목 수)
profile_bonus = np.zeros(len(df_skin))
if gender != '성별 선택':
    profile_bonus += df_skin['Gender'].eq(gender).to_numpy() * 0.05
if age != '연령대 선택':
    profile_bonus += df_skin['Age'].eq(age).to_numpy() * 0.05
if skin_type != '피부 타입 선택':
    profile_bonus += df_skin['Skin Type'].eq(skin_type).to_numpy() * 0.05

final_scores = similarities + profile_bonus
selected_index = int(final_scores.argmax())        # 최종 1위 상담 행
selected_row = df_skin.iloc[selected_index]

# ── Review Mode ─────────────────────────────────────────────────
similarities = linear_kernel(
    tfidf_reviews.transform([cleaned]),
    tfidf_matrix_reviews,
)[0]
ranked_indices = similarities.argsort()[::-1]
# score > 0 인 것만, product_name 중복 제거, 상위 3개
linear_kernel()은 이미 정규화(L2 norm)된 TF-IDF 벡터에 대해 내적만 계산하여 코사인 유사도와 동일한 결과를 더 빠르게 반환합니다.

Expert Mode vs Review Mode — 코사인 유사도 적용 방식

🔬 Expert Mode 전문가 상담 데이터 기반

항목내용

⭐ Review Mode 리뷰 기반

항목내용

Expert Mode 프로필 보너스

코사인 유사도 점수에 사용자 프로필(성별·연령대·피부타입) 일치 여부에 따라 보너스를 더합니다. 텍스트 유사도가 비슷할 때 사용자 특성에 맞는 상담 사례가 우선 선택되도록 합니다.

조건보너스최대 누적
성별 일치 (Gender)+0.05+0.15
연령대 일치 (Age)+0.05
피부타입 일치 (Skin Type)+0.05
# 예시: 20대 여성, 건성 피부 입력 시
# A 상담 사례: 여성 ✓, 20대 ✓, 건성 ✓ → +0.15
# B 상담 사례: 성별 불일치, 연령 불일치, 건성 ✓ → +0.05
final_A = 0.87 + 0.15  # = 1.02
final_B = 0.84 + 0.05  # = 0.89
# 코사인 유사도는 비슷하지만 A가 최종 1위
