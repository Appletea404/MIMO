KoNLPy란?

KoNLPy(Korean Natural Language Processing in Python)는 파이썬에서 한국어 형태소 분석을 수행하는 오픈소스 라이브러리입니다. 내부적으로 여러 분석기를 지원하며, 본 시스템에서는 Okt(Open Korean Text) 분석기를 사용합니다.


사용자가 입력한 자유형식의 한국어 문장을 의미 있는 단어 단위로 분해하여, 이후 TF-IDF 벡터화에 적합한 형태로 전처리하는 역할을 담당합니다.

핵심 역할: "건성 피부에 좋은 수분크림 추천해줘" → 의미 단어 토큰 추출 → TF-IDF 입력 텍스트 생성

실제 구현 코드 (web_app.py)

from konlpy.tag import Okt
import re

okt = Okt()

# stopwords.csv 로드
df_stopwords = pd.read_csv('datasets/stopwords.csv')
stopwords = df_stopwords['stopword'].tolist()

def preprocess_input(text):
    # ① 한글 이외 문자 제거
    text = re.sub('[^가-힣]', ' ', text)

    # ② 품사 태깅 (stem=True: 어간 추출)
    tokens = okt.pos(text, stem=True)

    # ③ 명사·동사·형용사 필터링 + 불용어 제거
    words = [word for word, part in tokens
             if part in ['Noun', 'Verb', 'Adjective']
             and len(word) > 1
             and word not in stopwords]

    return ' '.join(words)  # TF-IDF 입력용 문자열

stem=True 옵션을 사용하면 단어를 기본형(어간)으로 변환합니다. 같은 의미의 활용형들이 동일한 토큰으로 처리되어 매칭 정확도가 높아집니다.
