📊 TF-IDF

Term Frequency-Inverse Document Frequency · 텍스트 벡터화 · 단어 중요도 가중치

TF-IDF란?

TF-IDF(Term Frequency-Inverse Document Frequency)는 문서 내 단어의 중요도를 수치화하는 기법입니다. 단순 단어 빈도(TF)에 문서 전체에서의 희소성(IDF)을 곱하여 "이 문서에서만 자주 등장하는 핵심 단어"에 높은 가중치를 부여합니다.


본 시스템에서는 사용자 입력 쿼리와 전문가 상담 데이터의 각 질문을 TF-IDF 벡터로 변환하여, 이후 코사인 유사도 계산의 입력값으로 사용합니다.

핵심 역할: 전처리된 텍스트 → 수치 벡터 → 코사인 유사도 계산 가능한 형태

수식

TF (Term Frequency) — 단어 빈도

sublinear_tf(t, d) = 1 + log(tf(t, d))
현재 모델은 TfidfVectorizer(sublinear_tf=True)를 사용합니다.

IDF (Inverse Document Frequency) — 역문서 빈도

IDF(t) = log ((1 + N) / (1 + df(t))) + 1
sklearn 기본 smooth_idf=True 기준 · N = 전체 문서 수 · df(t) = 단어 t가 등장하는 문서 수

TF-IDF 최종 가중치

TF-IDF(t, d) = sublinear_tf(t, d) × IDF(t), 이후 L2 정규화
→ 해당 문서에서 의미 있게 등장하고, 다른 문서에서는 덜 흔할수록 높은 값

실제 구현 코드 (web_app.py)

import pickle
from scipy.io import mmread
from sklearn.metrics.pairwise import linear_kernel

# 서버 시작 시 1회 로드
tfidf_matrix = mmread('models/Tfidf_skin_data_rebuild.mtx').tocsr()
with open('models/tfidf_rebuild.pkl', 'rb') as f:
    tfidf = pickle.load(f)   # TfidfVectorizer 객체

# 추천 요청 시마다 실행
cleaned = preprocess_input(user_input)          # Okt 전처리
query_vec = tfidf.transform([cleaned])           # 쿼리 벡터화 (1×n)
similarities = linear_kernel(query_vec, tfidf_matrix)[0]  # 코사인 유사도
